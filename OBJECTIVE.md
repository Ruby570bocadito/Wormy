# 🎓 Proyecto de Investigación: ML Network Worm

## Objetivo Académico

Este proyecto fue desarrollado como trabajo de investigación para el Instituto de Ciberseguridad con los siguientes objetivos:

1. **Investigar la propagación automática de malware** - Estudio académico de cómo los gusanos сетевые se propagan en redes corporativas
2. **Aplicar Machine Learning a la seguridad** - Demostrar cómo los agentes de RL pueden optimizar la selección de objetivos
3. **Demostrar vulnerabilidades** - Mostrar la importancia de mantener sistemas actualizados y seguros
4. **Desarrollar herramientas defensivas** - Los mismos conceptos pueden usarse para detección y respuesta

## Contexto Educativo

Este proyecto forma parte de un trabajo de investigación sobre:
- Técnicas de propagación de malware
- Redes neuronales para toma de decisiones en ciberseguridad
- Análisis de vulnerabilidades en redes empresariales

## Enfoque del Proyecto

El valor académico de este proyecto está en:
1. **Arquitectura del sistema** - Cómo se diseña un sistema de propagación complejo
2. **Agente de RL** - Implementación de aprendizaje por refuerzo para selección de objetivos
3. **Mecanismos de seguridad** - Kill switches, auto-destruct, geofencing
4. **Análisis de vulnerabilidades** - Estudio de CVEs conocidos
